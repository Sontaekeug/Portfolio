# 미사일 피하기 게임(DodgeMissile, feat. FireBase)

## 소개
과거 곰플레이어 이스터에그로 유명했던 미사일 피하기 게임을 재현, FireBase 서비스를 통한 인터넷 랭킹 시스템 구축

## 설 명
- 미사일을 피해 우주선을 오래 생존시키는 게임
- 시간이 지날수록 미사일 난이도 상승(속도, 미사일 수 증가)
- 랭킹시스템을 통한 오랜 시간 피한 사람들의 랭킹 표시

## 실행 방법
1. python 코드 파일은 작성 코드 참고용으로 업로드하였으며, FireBase 관련 정보(Product ID, API Key, API URL, JSON 파일 등)를 모두 삭제하여 실행되진 않습니다.
2. 아래 URL을 통해 exe 파일 다운로드 및 실행 필요
   [다운로드 링크](https://drive.google.com/file/d/1uZQ7LT1jRo-HGUUXdzmMM-LPxL2yx_TP/view?usp=drive_link)

## 주의사항
- 파이썬 코드만으로는 실행할 수 없습니다.
- 반드시, 파일 실행 방법 안내 란의 다운로드 링크를 통해 EXE 파일을 다운받아 실행해주세요.
